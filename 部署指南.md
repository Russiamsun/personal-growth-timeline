## 手动部署步骤

### 1. 创建GitHub仓库

1. 登录GitHub：https://github.com
2. 点击右上角的 "+" 号，选择 "New repository"
3. 填写仓库信息：
   - Repository name: `personal-growth-timeline`
   - Description: 个人成长时间轴网站
   - 选择 Public（公开仓库，GitHub Pages免费）
   - 不要勾选 "Add a README file"
   - 不要勾选 "Add .gitignore"
   - 不要勾选 "Choose a license"
4. 点击 "Create repository"

### 2. 添加远程仓库并推送代码

创建仓库后，GitHub会显示一些命令，你需要执行：

```powershell
# 添加远程仓库（替换为你的实际仓库URL）
git remote add origin https://github.com/Russiamsun/personal-growth-timeline.git

# 将当前分支重命名为main
git branch -M main

# 推送代码到GitHub
git push -u origin main
```

### 3. 启用GitHub Pages

1. 进入仓库页面：https://github.com/Russiamsun/personal-growth-timeline
2. 点击 "Settings"（设置）
3. 在左侧菜单找到 "Pages"
4. 在 "Build and deployment" 部分：
   - Source: 选择 "GitHub Actions"
5. GitHub Actions会自动开始部署你的网站

### 4. 等待部署完成

- 部署通常需要1-3分钟
- 在Actions页面可以看到部署进度：https://github.com/Russiamsun/personal-growth-timeline/actions
- 部署完成后，你的网站将可以通过以下URL访问：
  **https://Russiamsun.github.io/personal-growth-timeline/**
