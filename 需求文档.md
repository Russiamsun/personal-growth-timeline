# Emma's Growth Portfolio 需求文档

## 一、项目概述

**项目名称**: Emma's Growth Portfolio
**项目定位**: 记录个人成长的Portfolio网站
**核心价值**: 记录Emma在AI时代的成长历程，展示真实世界经历、思考过程和成长反思
**目标用户**: 个人成长记录与展示

---

## 二、设计理念

遵循"第一阶段（现在｜9岁）建立成长观察"的设计理念：

- **不复杂**：只保留三个核心栏目
- **真实**：每个栏目都承载真实的好奇心、真实的成长、真实的反思
- **专业**：简洁专业的设计风格，去除内部指导性文字
- **差异化**：每个板块独特的设计风格

---

## 三、三大核心板块

### 3.1 Real World Experiences（真实世界经历）

| 活动类型 | 说明 |
|---------|------|
| 公益活动 | 参加公益项目、志愿服务 |
| 科学馆 | 科学中心、博物馆参观 |
| 城市探索 | 城市文化、历史探访 |
| AI体验 | AI工具使用、AI项目参与 |
| 社会观察 | 社会现象观察、调研 |

**核心数据字段**：
- 活动标题、描述、详细内容
- 活动日期、地点
- 活动类型（charity、science-museum等）
- 照片（支持多张）
- 标签

### 3.2 Questions I'm Thinking About（正在思考的问题）

**核心特点**：
- genuine curiosity（真实好奇心）
- 体现思考深度
- 来自真实经历的思考

**核心数据字段**：
- 问题内容
- 思考过程（可选）
- 日期
- 标签

### 3.3 Reflection Journal（成长反思）

**核心特点**：
- 简短真实（甚至一句话都行）
- 来自真实感悟
- 比奖状高级

**核心数据字段**：
- 反思内容
- 日期
- 标签

## 四、功能需求

### 4.1 CRUD功能（优先级：P0）

| 功能 | 活动板块 | 问题板块 | 反思板块 |
|------|---------|---------|---------|
| 创建 | ✓ | ✓ | ✓ |
| 编辑 | ✓ | ✓ | ✓ |
| 删除 | ✓ | ✓ | ✓ |
| 查看 | ✓（详情页） | ✓（列表页） | ✓（列表页） |

### 4.2 数据管理功能（优先级：P0）

| 功能 | 说明 |
|------|------|
| 数据持久化 | localStorage存储，刷新不丢失 |
| 自动同步 | 新增活动自动出现在时光轴 |
| 数据加载 | 从localStorage或初始数据加载 |

### 4.3 时光轴功能（优先级：P1）

| 功能 | 说明 |
|------|------|
| 活动展示 | 显示所有activities数据 |
| 日期排序 | 按日期降序排列 |
| 时间线设计 | 左右交替的现代化时间线 |
| 详情导航 | 点击卡片跳转到活动详情 |

### 4.4 照片展示（优先级：P2）

| 功能 | 说明 |
|------|------|
| 照片占位 | 当前使用picsum.photos占位图片 |
| 多张照片 | 每个活动支持6张照片 |
| 照片网格 | 2x3网格布局展示 |
| 照片说明 | 每张照片有caption说明 |

## 七、数据模型

### 7.1 Activity类型

```typescript
{
  id: string;
  type: 'charity' | 'science-museum' | 'city-explore' | 'ai-experience' | 'social-observation';
  title: string;
  description: string;
  content: string; // HTML富文本
  date: string; // YYYY-MM-DD
  location?: string;
  photos: Photo[];
  tags: string[];
  createdAt: string;
  updatedAt: string;
}
```

### 7.2 Question类型

```typescript
{
  id: string;
  question: string;
  thoughts?: string;
  date: string;
  tags?: string[];
  createdAt: string;
  updatedAt: string;
}
```

### 7.3 Reflection类型

```typescript
{
  id: string;
  content: string;
  date: string;
  tags?: string[];
  createdAt: string;
  updatedAt: string;
}
```

### 7.4 Photo类型

```typescript
{
  id: string;
  url: string;
  order: number;
  caption?: string;
  uploadedAt: string;
}
```
