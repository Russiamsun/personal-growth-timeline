# 灵’s Future Human Portfolio 网站设计理念 (1)

**未来三年，**

我们的网站会怎么慢慢长出来。

# **第一阶段（现在｜9岁）**

# **建立“成长观察”**

网站不要复杂。

甚至可以只有：

# **首页：**

# **Emma｜Growing Up in the AI Era**

（成长于AI时代）

下面一句话：

一个关于好奇心、真实世界与未来成长的长期观察计划。

就已经很高级了。

# **目前只放3个栏目：**

# **1. Real World Experiences**

真实世界经历

放：

- 公益活动
- 科学馆
- 城市探索
- AI体验
- 社会观察

# **2. Questions I’m Thinking About**

我正在思考的问题

这个特别重要。

例如：

- 为什么有的小朋友长期住院？
- AI会不会让人孤独？
- 科技能不能帮助儿童？

顶级学校会特别喜欢这种：

# **genuine curiosity（真实好奇心）**

# **3. Reflection Journal**

成长反思

不用长。

甚至一句话都行。

例如：

今天我第一次觉得，

有些小朋友比我更需要帮助。

这种东西，

其实比奖状高级很多。

# **第二阶段（10-11岁）**

# **开始出现“方向感”**

Narrative开始慢慢形成。

例如：

# **她持续关注：**

- 儿童
- 科技
- AI
- 公益
- 教育公平

网站开始增加：

# **Projects（项目）**

例如：

- AI陪伴小项目
- 儿童采访
- Storytelling
- 小研究

# **第三阶段（12岁）**

# **Narrative开始成型**

这时候，

你会突然发现：

# **她已经不是“参加活动”。**

而是：

# **开始形成自己的世界理解。**

例如：

她长期关注儿童与AI。

她开始思考：

“科技如何让长期住院儿童减少孤独感？”

这个时候，

网站会开始非常有力量。
